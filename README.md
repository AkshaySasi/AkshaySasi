# 🚀 Featured Projects

## 🧠 Neurogram – Cognitive Memory Architecture for AI Agents
> A biologically inspired memory system for autonomous AI agents.

**Features**

- Semantic, Episodic & Procedural memory types
- Memory consolidation & deduplication
- Ebbinghaus forgetting curve
- Importance scoring
- SQLite-based local storage
- LangChain integration

🔗 https://github.com/AkshaySasi/Neurogram

---

## 🔐 GlassLM – Privacy-Preserving LLM Middleware
Secure prompt processing layer that **protects sensitive information before sending prompts to LLM APIs.**

**Key Features**

- Client-side prompt sanitization
- Reversible placeholder tokenization
- Chrome extension
- Developer SDKs (`pip` + `npm`)

🔗 https://github.com/AkshaySasi/GlassLM

---

## 🏥 Delumie – Local AI Healthcare Companion
Privacy-first healthcare AI assistant running **entirely on local LLM inference.**

**Highlights**

- Ollama-powered local AI
- Modular health guidance system
- Personalized diet & wellness insights
- Zero cloud dependency

🔗 https://github.com/AkshaySasi/Delumie
